# Proxy

![proxy](http://www.imageup.ru/img130/2765966/tab_proxy.png)

1) Список прокси-серверов.

2) Name - Имя прокси сервера.

3) Address - IP-адрес прокси-сервера.

4) Port - Порт прокси-сервера.

5) SOCKS5 - Использование SOCKS5 проксей (с авторизацией).

6) Account - Аккаунт для авторизации на прокси-сервере.

7) Password - пароль для авторизации на прокси-сервере.

8) Encrypt password - Включить шифрование пароля (временно не работает).

9) Show password - Показать пароль вместо звездочек.

10) Add new - Добавить новый прокси-сервер с введенными данными.

11) Save selected - Сохранить выбранный прокси-сервер.

12) Remove selected - Удалить выбранный прокси-сервер.
