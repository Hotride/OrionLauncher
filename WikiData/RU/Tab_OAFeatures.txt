# OA Features

![oafeatures](http://www.imageup.ru/img130/2765968/tab_oafeatures.png)

1) Список регулируемых опций:

Corpses autoopen - Автоматическое открытие трупов.

Stealth steps counter - Счетчик шагов в стелсе.

Sound filter - Фильтр звуков.

Light filter - Фильтр смены дня и ночи.

Weather filter - Фильтр погоды.

Season filter - Фильтр сезонов года.

No death screen - Регулировка появления черного экрана от смерти (серый экран при этом останется).

Auto open doors - Автоматическое открытие дверей.

Tracker - Отображать квестовую стрелку после смерти, которая укажет направление, в котором лежит труп.

Reserverd - Зарезервировано.

Reserved - Зарезервировано.

Fast rotation - Быстрый поворот персонажа.

Recurse search - Рекурсивный поиск в контейнерах (поиск объектов и в подсумках, при условии, что они были открыты).

Autostart - Выполнение функции Autostart (из скриптов) при входе в игру.

Ignore staminacheck - Игнорирование клиентом проверки стамины при проходе через персонажей.

Party agent - Использование агента групп.

Shop agent - Использование агента магазина.

Tray icon - Возможность сернуть ассист в трэй.

2) Список регулируемых групп скриптовых команд:

Information - Команды Info/InfoTile/InfoMenu.

Text window - Команды по работе с текстовым окном.

Containers search - Команды поиска объектов в контейнерах FindType/FindList.

Ground search - Команды поиска объектов на земле FindType/FindList.

Moving - Команды перемещения персонажа в мире.

Resend - Команда синхронизации с сервером.

Journal - Команды для работы с журналом.

Rename mounts - Команда переименования петов RenameMount.

Buy - Команда автопокупки.

Sell - Команда автопродажи.

Hide - Команда для скрытия объектов с экрана Hide.

Move items - Команды для перемещения предметов MoveItem/Drop/DropHere.

Morph - Команда визуального превращения в существо Morph.

Emote actions - Команды для воспроизведения эмоций EmoteAction.

Find/Ignore types - Возможность использовать поисковые команды FindType и команды игнорирования объектов Ignore/IgnoreReset.

Find arrays - Команды для работы со списками поиска FindList.

Client options - Команды для получения и установки опций клиента.

Corpse search - Команды для обыска трупов FindType/FindList.

Extended menu - Команды расширенного функционала для работы с меню.

Buffs - Команды для работы с баффами.

Trading - Команды для работы с окном торговли.

Friend list - Команды для работы со списком друзей.

Enemy list - Команды для работы со списком врагов.

Font color - Команды для смены цвета текста.

Extended target tile - Команды для автоопределения типа тайлов при использовании прицелов.

3) Output code for emulator - Для какого эмулятора генерировать код. Код генерируется при переключении типа.

4) Сгенерированный код для регулировки опций и команд ассистента.
