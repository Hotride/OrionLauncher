# Server

![server](http://www.imageup.ru/img130/2765964/tab_server.png)

1) Список серверов.

2) Orion path - Путь к папке с Орионом (и файлами УО). При изменении/указании пути производится проверка обновлений для файлов в указанной папке, если обновления есть - перекидывает на вкладку обновлений.

3) Указать путь к папке с Орионом.

4) Configure client version - Настроить конфигурацию Ориона (необходимо что бы в указанной папке с УО был файл ConfigurationEditor.exe). Если файла Client.cuo в папке нет - текст кнопки будет красным.

5) Check updates - Автоматическая проверка обновлений для файлов из выбранной папки с УО.

6) Command line - Дополнительные команды для старта клиента.

7) Auto login - Выставить галочку в клиенте Auto Login.

8) Save password - Выставить галочку в клиенте Save Password.

9) Save aero (for assistant) - Не убирать aero в ассисте.

10) Run UOAM - Запускать UO Auto Map вместе с клиентом. Карта должна быть в папке с УО. Запускает данный файл: <UODir>/Map/EnhancedMap.exe.

11) Fast login - Автоматически входить в игру при запуске клиента.

12) Launch - Запустить клиент с выбранными параметрами.

13) Close after launch - Закрывать лаунчер после запуска клиента.

14) Name - Имя конфигурации сервера.

15) Address - IP-адрес сервера. Вводится в формате: IP,Port например: uo.drw.ru,2593

16) Account - Аккаунт на сервере.

17) Password - Пароль от аккаунта.

18) Encrypt password - Включить шифрование аккаунтов/паролей (временно не работает).

19) Use proxy - Использовать прокси-сервер при подключении.

20) Show password - Отображать пароль.

21) Список сохраненных прокси серверов.

22) Add new - Добавить новую конфигурацию с введенными данными.

23) Save selected - Сохранить выбранную конфигурацию.

24) Remove selected - Удалить выбранную конфигурацию.
