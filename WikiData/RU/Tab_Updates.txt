# Updates

![updates](http://www.imageup.ru/img130/2765967/tab_updates.png)

1) Список файлов, доступных для обновления.

2) Прогресс скачивания и установки обновлений.

3) Check updates - запрос данных о обновлениях.

4) Apply updates - применить обновления (если они есть).

5) Download Orion Assistant with libraries - скачать и установить последнюю версию ассистента вместе со всеми необходимыми библиотеками.

6) Download Orion Launcher with libraries - скачать и установить последнюю версию лаунчера вместе со всеми необходимыми библиотеками.
